# Python-Tic-Tac-Toe
My first working python game made from scratch.

This was done for my Augusta Technical College class, "CIST 1305: Program Design & Development".
Upon launching you are prompted to input a number between 1 and 9 to make your move on the board.
The "computer" opponent is designed to always make its winning move if it is 1 move away from winning the game.
#### I've programmed in safeguards to: 
- Prevent the player from entering a number outside of the required range (1:9).
- Prevent the player from entering a non-numerical value.
